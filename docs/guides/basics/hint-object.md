# Hint object

You can use a **Hint object** in the editor. The object is ideal if you need to specifically highlight text, such as a notification, warning, or hint. Besides the text, you can also paste an icon that can add emotion to the message. Add a hint on the object menu.

## How to add a hint? [#](#b3510665761_43)

-   Click on the object menu and select **Hint** or use the hotkey **cmd/ctrl** + **⌥/alt** \+ u.

![How to add a hint?](https://app.nimbusweb.me/box/attachment/5452817/8bwvsm6mp4w5pggfgx3y/ySs9MgOslxVK2hRu/Group%20191%20(1).png)

## How to change the hint icon? [#](#b3510665761_463)

-   Click on the **icon**.
-   Select **another icon**.

![How to change the hint icon?](https://app.nimbusweb.me/box/attachment/5452817/8bwvsm6mp4w5pggfgx3y/EIBItBCcOlQytkor/Group%20192%20(1).png)

## How to add a border? [#](#b3510665761_592)

-   Click on the **menu**.
-   Select **Border Color**.

![How to add a border?](https://app.nimbusweb.me/box/attachment/5452817/8bwvsm6mp4w5pggfgx3y/VtgqieYnnG2Khtr2/Group%20267.png)

## How to add a colored line? [#](#b2746284124_182)

-   Click on the **menu**.
-   Select **Line color.**
