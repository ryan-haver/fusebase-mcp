# Task Board Guide

In [FuseBase](https://thefusebase.com/), you can create not only the usual task lists but also full-fledged Kanban Boards using the Task Board object. You can create an object through the menu of objects in the editor –

![You can create an object through the menu of objects in the editor](https://thefusebase.com/wp-content/uploads/2023/01/Task_Board_1.png)

By default, two columns are created in the board, but you can create new ones through the Add column option –

![By default, two columns are created in the board, but you can create new ones through the Add column option](https://thefusebase.com/wp-content/uploads/2023/01/Task_Board_2.png)

## How do I rename the board? [#](#0-toc-title)

By default, the board has a standard name, but you can rename it. To do this, simply click on the name of the board and specify the desired name –

![By default, the board has a standard name, but you can rename it. To do this, simply click on the name of the board and specify the desired name](https://thefusebase.com/wp-content/uploads/2023/01/Task_Board_3.png)

## How do I rename a column name? [#](#1-toc-title)

Hover over the column name and click on the edit icon –

![Hover over the column name and click on the edit icon](https://thefusebase.com/wp-content/uploads/2023/01/Task_Board_5.png)

Then enter a new column name and press Enter to confirm –

![Then enter a new column name and press Enter to confirm](https://thefusebase.com/wp-content/uploads/2023/01/Task_Board_5-1.png)

## How do I add labels/descriptions/date, etc. to a task? [#](#2-toc-title)

Click the name of the task to open it in a detailed view. Next, you can add a description, labels, date, etc. to the task –

![Click the name of the task to open it in a detailed view. Next, you can add a description, labels, date, etc. to the task](https://thefusebase.com/wp-content/uploads/2023/01/Task_Board_6-1024x559.png)

## How do I delete a column? [#](#3-toc-title)

Click on the column’s menu icon and select Delete –

![Click on the column's menu icon and select Delete](https://thefusebase.com/wp-content/uploads/2023/01/Task_Board_7.png)

## How can I delete a task? [#](#4-toc-title)

Open the task in detail view and select Delete task in the task menu –

![Open the task in detail view and select Delete task in the task menu](https://thefusebase.com/wp-content/uploads/2023/01/Task_Board_8-1024x433.png)

## Can I move columns and tasks within the board? [#](#5-toc-title)

Yes, of course. You can:

-   Swap columns
-   Reorder tasks in columns
-   Transfer tasks between columns

To do this, click on a column or task and keep the mouse button pressed while doing the necessary action –

![](https://thefusebase.com/wp-content/uploads/2023/11/TASKBOARDGUIDE-1024x512.gif)

## Is it possible to convert the board column to a regular task list? [#](#6-toc-title)

Yes, you need to select Convert to the task list in the column menu to do this –

![Yes, you need to select Convert to the task list in the column menu to do this](https://thefusebase.com/wp-content/uploads/2023/01/Task_Board_10.png)
