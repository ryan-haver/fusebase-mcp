# How to use templates in FuseBase

## Are there any pre-made templates in FuseBase? [#](#b2601203255_149)

-   Yes, [FuseBase](https://thefusebase.com/) has a [gallery of templates](https://thefusebase.com/templates/) for different categories

![Nimbus Note has a gallery of templates for different categories](https://app.nimbusweb.me/box/attachment/4465538/6yutrkcsqw3vlrkqi291/cBbB7XpHk3ZMvTuX/Group%2013.png)

## How to create a page using a template [#](#b2601203255_297)

-   Press Create a new page and press **Templates**
-   Select the desired template and press **Use template**.
-   A page will be created based on the template selected.

![How to create a page using a template](https://app.nimbusweb.me/box/attachment/4465538/6yutrkcsqw3vlrkqi291/M2XrCyZ2Vf2aOIz2/11_4%20(1).gif)

## How to duplicate a template page [#](#b1745312559_10)

-   To make a copy of a template in your workspace, right click on the page and choose **Duplicate**.

![How to duplicate a template page](https://app.nimbusweb.me/box/attachment/4465538/6yutrkcsqw3vlrkqi291/AYlG4wwcdMn7x62H/Group%2010%20(2).png)

## How to copy a table from a template [#](#b1745312559_201)

![How to copy a table from a template](https://app.nimbusweb.me/box/attachment/4465538/6yutrkcsqw3vlrkqi291/4KUR6mxPIt24kvAl/preview%20(1).gif)

-   To copy a table of a template click on the menu button and choose **Copy table**
-   Then you will be able to paste it into other pages via **Ctrl** + **V** or right-click menu, then **Paste**

## How to save a template from public page [#](#b1745312559_489)

-   Click on the **Save** button and select the workspace where you would like to save the template

![How to save a template from public page](https://app.nimbusweb.me/box/attachment/4465538/6yutrkcsqw3vlrkqi291/z3vMTzF1cI4pMUnf/2.png)

Then go to FuseBase web client ( https://thefusebase.com/client ) and you can find saved template.
