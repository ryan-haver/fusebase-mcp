# Dropdown List

With the Dropdown object, you can create a sheet with a list of options to choose from, depending on your needs.

## Creating a List [#](#0-toc-title)

You can create a list through the objects menu – **Dropdown List**.

![Creating a List. Image by FuseBase](https://thefusebase.com/wp-content/uploads/2023/03/preview-5.png)

Next, you need to make a list of options that will be on the sheet. There can be several options in the list, but only one can be chosen.

![Dropdown list. Gif by FuseBase](https://thefusebase.com/wp-content/uploads/2023/03/Dropdown-list.gif)

To select another option from the list, click on **the** **list item**.

![List Item. Image by FuseBase](https://thefusebase.com/wp-content/uploads/2023/03/List-item.png)

Choose the desired option from the list.

![The Desired Option. Image by FuseBase](https://thefusebase.com/wp-content/uploads/2023/03/The-desired-option.png)

## Changing List Items (renaming and color changing) [#](#1-toc-title)

You can change individual list items, such as renaming or changing color. You can do this via **the list item menu**.

![Individual List Items. Image by FuseBase](https://thefusebase.com/wp-content/uploads/2023/03/Individual-List-Items.png)

You can rename the option name or change its color.

![The Option Name. Image by FuseBase](https://thefusebase.com/wp-content/uploads/2023/03/The-option-name.png)

You can also change the style of the list.

![The Style of the List. Image by FuseBase](https://thefusebase.com/wp-content/uploads/2023/03/The-Style-of-the-List.png)

## Deleting a List Item [#](#2-toc-title)

To delete a list item, you need to select **the Delete option** in the item menu.

![Deleting the List Item. Image by FuseBase](https://thefusebase.com/wp-content/uploads/2023/03/Deleting-List.png)

## Deleting the List [#](#3-toc-title)

You can delete the entire list just like any other object from the object menu.

![Deleting the List. Image by FuseBase](https://thefusebase.com/wp-content/uploads/2023/03/Deleting-the-List.png)
