# How to duplicate templates and tables

## How to duplicate a template page [#](#b3359821425_111)

-   To make a copy of a template in your workspace, right click on the page and choose **Duplicate**.

![How to duplicate a template page](https://app.nimbusweb.me/box/attachment/4465543/gaeb94l6a5tk03glkm6i/xSLW5JfkuXBbcRNF/Group%2010%20(2).png)

## How to copy a table from a template [#](#b1379747865_0)

![How to copy a table from a template](https://app.nimbusweb.me/box/attachment/4465543/gaeb94l6a5tk03glkm6i/sZ6xwiGlMxPLexRs/preview%20(1).gif)

-   To copy a table of a template click on the menu button and choose **Copy table**
-   Then you will be able to paste it into other pages via **Ctrl** + **V** or right-click menu, then **Paste**

## How to save a template from public page [#](#b3347856364_39)

-   Click on the Save button and select the workspace where you would like to save the template

![How to save a template from public page](https://app.nimbusweb.me/box/attachment/4465543/gaeb94l6a5tk03glkm6i/CKzJQpNFelpdenwt/2.png)

Then go to FuseBase web client (https://thefusebase.com/client) and you can find saved template.
