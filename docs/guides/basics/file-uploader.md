# File Uploader

The File Uploader allows you to aggregate several files in one place with the ability to add a name and description to them. You can create an object through **the objects menu**.

![](https://thefusebase.com/wp-content/uploads/2023/04/image.png)

## Adding Files [#](#0-toc-title)

To add files, click on **Click to Upload**:

![](https://thefusebase.com/wp-content/uploads/2023/04/image-1.png)

After that, select the files you want. You can also Drag and Drop files. After selecting and successfully uploading, the files will appear in the list:

![](https://thefusebase.com/wp-content/uploads/2023/04/image-2.png)

## **Downloading Files** [#](#1-toc-title)

You can download the desired file by clicking on **the file name** or by clicking on **the Download File icon**:

![](https://thefusebase.com/wp-content/uploads/2023/04/image-3.png)

## **File Management** [#](#2-toc-title)

You can manage files through their menu.

![](https://thefusebase.com/wp-content/uploads/2023/04/image-4.png)

Here you can:

-   Rename the file;
-   Convert it to a regular file (outside the object);
-   Copy a link to a file;
-   Delete file.

![](https://thefusebase.com/wp-content/uploads/2023/04/image-5.png)

## Renaming an Object and Adding a Description [#](#3-toc-title)

You can also rename the object or add your own description. To do that, click on the appropriate option and in the field that appears type in the name you want, then press **Enter**.

![](https://thefusebase.com/wp-content/uploads/2023/04/image-6.png)

![](https://thefusebase.com/wp-content/uploads/2023/04/image-7.png)

## Banning Objects from Downloading Files [#](#4-toc-title)

If you want, you can forbid downloading any files from the object by clicking on **the object’s menu** and unchecking the Downloading checkbox:

-   For workspace users – banning downloads to the web client
-   Ban on downloading in public pages

![](https://thefusebase.com/wp-content/uploads/2023/04/image-8.png)

You can also enable a download ban for individual files in an object. You can do this through the file menu:

![](https://thefusebase.com/wp-content/uploads/2023/04/image-9.png)

## Deleting an Object [#](#5-toc-title)

If you want, you can delete the object with all its files. You can do this through the object menu:

![](https://thefusebase.com/wp-content/uploads/2023/04/image-10.png)

![](https://thefusebase.com/wp-content/uploads/2023/04/image-11.png)
