# Tags – Quick Guide

Tags give your pages context and meaning. The main advantage of tags is that you can add many of them to one page (unlike a folder). It’s easier to categorize them that way. Let’s say there’s a page titled “Vegetable Salad Recipe.” You can place it in the Food folder, but you can also add tags – vegetables, salad, tomatoes, cucumbers, and so on.

Using tags you can access pages in one click. Even if they are located in different folders, you can call them up them easily if they share the same tag.

## Adding a tag to a page [#](#0-toc-title)

To add a tag to a page, you need to do a few simple steps:

1) Open the desired page.

2) Click on Add tag.

3) Select a tag from the existing ones or type the name of a new tag and press Enter.

## How to view the list of all tags [#](#1-toc-title)

The list of all tags is displayed in the sidebar under Tags. Click on the desired tag to see the pages with the selected tag.

## How to remove a tag from a page [#](#2-toc-title)

Simply click the X button on the tag name.

## How to delete a tag from the workspace [#](#3-toc-title)

Hover over the tag name in the sidebar and click Delete.

## How to rename a tag [#](#4-toc-title)

Hover over the tag name in the sidebar and click Rename. Then enter the new tag name.

## How to create a tag without linking it to a page [#](#5-toc-title)

Click on the + next to Tags in the sidebar. In the popup that appears, enter the tag name and click Save.
