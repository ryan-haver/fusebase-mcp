# Images

FuseBase allows you to add pictures with various extensions (jpg, gif, png, etc.) to your page. You can also add files directly into tables, resize & move pictures, copy text from images and much more.

## Add image and drop / resize / move it [#](#b3734483443_386)

There are several ways to add a picture:

-   Select the picture icon on the panel or use **cmd/ctrl** + **shift** + **i**.
-   You can also drag and drop an image into your page.

-   Alternatively, you can insert the URL using **сtrl** + **v**.

### Add a picture into the table [#](#b2760204264_399)

To add a picture into the table:

-   Click the table icon or use **cmd/ctrl** + **⌥/alt** + **t**.
-   Select **Attachment** as the type of column or cell.
-   Upload a file from your computer or drag and drop the file.

## Image file menu [#](#b2150728477_0)

### Menu in the editor [#](#b2960557591_0)

Click the three dots next to an image to open the menu for the picture in the editor.

![Click the three dots next to an image to open the menu for the picture in the editor.](https://app.nimbusweb.me/box/attachment/4485409/7ydr5g05c11aya8la5kg/r2fJ87BRkkIUBY4c/4.gif)

#### Image alignment [#](#b2427682051_0)

Click on the three dots menu and select left / center / right / full width alignment.

#### Default alignment for images [#](#b2026113424_527)

All new images are center-aligned by default. However, it might be not convenient for all the users. You can choose the default image alignment settings that suit you best! ?

Just go to the Settings – Page Appearance and set the default image alignment you like!

#### Add block [#](#b2427682051_143)

To add a block, select the three dots menu and press **Add block**.

![](https://thefusebase.com/wp-content/uploads/2022/08/add-block-1024x614.png)

#### Convert to file [#](#b2427682051_252)

In FuseBase, you can convert images to files. Just click on the three dots menu and select **Convert to.**

#### Copy Link [#](#b2073141922_589)

You can copy a link to a block and paste it into another page. Clicking on the block’s link will direct you to the block’s location in the original page.

#### Get text from image [#](#b2073141922_1211)

In FuseBase, you have the option to grab text from images. This text can later be copied and pasted wherever you need.

Image-to-text recognition is a powerful feature that enables searching for text within images. You may have always wanted to extract the text from some of your scanned photos or documents. The OCR capabilities in FuseBase can make this a reality for you.

#### Replace [#](#b2427682051_380)

To replace the picture with another click on the three dots menu and select **Replace**.

![](https://thefusebase.com/wp-content/uploads/2022/08/replace-1024x612.png)

#### Rotate [#](#b2427682051_528)

To rotate the picture choose **Rotate**.

#### Lifehack [#](#b2015698834_10)

When you open a picture in full-screen mode, you can rotate it, zoom in or out, and print it. To open a picture in full-screen mode, double-click on it.

![When you open a picture in full-screen mode, you can rotate it, zoom in or out, and print it. To open a picture in full-screen mode, double-click on it.](https://app.nimbusweb.me/box/attachment/4485409/7ydr5g05c11aya8la5kg/JMEnrXJTZTuzWIqM/Comp_1%201828%20center.gif)

#### Resize images [#](#b756089151_40)

-   Drag the upper-right corner and **resize** (all proportions will be preserved).

![Drag the upper-right corner and resize (all proportions will be preserved).](https://app.nimbusweb.me/box/attachment/4485409/7ydr5g05c11aya8la5kg/beFzv3Ly0Mzcd0xc/Group%20142%20(1).png)

-   For free resizing, hold the **ALT key** and change the size using the bottom-right chip.

![For free resizing, hold the ALT key and change the size using the bottom-right chip.](https://app.nimbusweb.me/box/attachment/4485409/7ydr5g05c11aya8la5kg/cHfuKFVE0qkTpsPq/Group%20143.png)

**☝Note: To return the image to its original position, click on the block menu and select the Default size icon.**

![To return the image to its original position, click on the block menu and select the Default size icon.](https://app.nimbusweb.me/box/attachment/4485409/7ydr5g05c11aya8la5kg/5doyjCoY5YKVQTX6/Group%20179.png)

#### Set as preview [#](#b3284289719_191)

To set the picture as a preview, select **Set as preview**.

**☝** **Note: Preview pictures are synced between clients. This means if you have installed previews in the web version, then the same preview will be in mobile clients.**

#### Layout [#](#b3278002868_30)

**Shadow effect for image**

In FuseBase, you can highlight not only text, but also pictures and video. This is possible with the help of Shadow effect.

-   Click on the picture menu.
-   Select **Layout**.
-   Activate **Shadow for image.**
