# Exporting a Table to CSV – Quick Guide

In Fusebase, you can export any table to a CSV file.

1) Open the desired database and navigate to the table you want to export.

2) Click on the menu arrow and select Export.

![](https://thefusebase.com/wp-content/uploads/2026/01/image-73-1024x506.png)

3) Next, specify the column delimiter. By default, this is usually a comma, and we recommend keeping this setting.

![](https://thefusebase.com/wp-content/uploads/2026/01/image-74-1024x610.png)

4) Next, click Export and wait for the export process to complete.
