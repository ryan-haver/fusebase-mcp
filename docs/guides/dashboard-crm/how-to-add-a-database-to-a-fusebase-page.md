# How to add a database to a Fusebase page

You can work with databases not only in the Databases section, but also directly on a page.

To do this, follow a few simple steps:

1) Go to the desired page or create a new one.

2) Click on the object menu and select Database.

![](https://thefusebase.com/wp-content/uploads/2025/12/image-4-1024x746.png)

3) Next, you can create a new database or choose from existing ones. You can select an entire database, individual tables, or even a specific view.

![](https://thefusebase.com/wp-content/uploads/2025/12/image-5-1024x704.png)

If you need to insert the entire database, select it from the Database list and click Add database. If you want to insert only a table, select the table from the list; if you need a specific view, select the desired view.

![](https://thefusebase.com/wp-content/uploads/2025/12/image-6-1024x710.png)

## How to create a new database [#](#0-toc-title)

Click on the create database icon in the object menu.

![](https://thefusebase.com/wp-content/uploads/2025/12/image-7-1024x492.png)

You can also select the Create new database option in the database selection window.

![](https://thefusebase.com/wp-content/uploads/2025/12/image-8-1024x698.png)

## How to remove a table from a page [#](#1-toc-title)

You need to click on the block menu and select Delete. Please note that the table will only be removed from the page, but will remain in the Databases section. Accordingly, if you want to delete the database completely, you need to go to Databases and delete it from there.

![](https://thefusebase.com/wp-content/uploads/2025/12/image-9-1024x390.png)

It is also important to note that if you delete a database, it will not be automatically removed from the page. A placeholder will be added to the page with a message indicating that the database is unavailable, and you can remove this placeholder.
