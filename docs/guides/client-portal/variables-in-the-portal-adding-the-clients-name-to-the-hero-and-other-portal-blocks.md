# Variables in the portal – adding the client’s name to the Hero and other portal blocks.

Variables are special text elements in the portal that are replaced with various preset values. For example, this could include the client’s name, company name, current date, and so on.

In this guide, I will explain how to add client names for greetings in the Hero section. Using client names is highly beneficial when creating a personalized portal.

First, you need to navigate to the **edit mode** of the Hero section in the portal.

![](https://thefusebase.com/wp-content/uploads/2024/12/image-2-1024x559.png)

Next, go to the **welcome message** editing area.

![](https://thefusebase.com/wp-content/uploads/2024/12/image-3-1024x559.png)

Let’s set it up so that the portal slogan displays the text: **Welcome, Client’s Name!** To do this, type **Welcome,** in the input field and click on the **X** icon to open the list of available variables. From the list, select **Client’s Name.**

![](https://thefusebase.com/wp-content/uploads/2024/12/image-4-1024x559.png)

![](https://thefusebase.com/wp-content/uploads/2024/12/image-5-1024x559.png)

The input will now include the client name variable, which will be replaced with the actual client’s name in the portal.

![](https://thefusebase.com/wp-content/uploads/2024/12/image-6-1024x559.png)

Next, click on **Save** and **Publish** to save the changes. You can add variables not only to the Hero section but also to almost any text input field in the blocks available within the portal.
