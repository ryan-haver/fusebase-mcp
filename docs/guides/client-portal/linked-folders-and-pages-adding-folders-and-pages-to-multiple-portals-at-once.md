# Linked Folders and Pages – Adding folders and pages to multiple portals at once

This feature allows you to link the same folders or pages across several portals. Any updates made to the content will automatically sync across all portals where these folders or pages are linked. It’s a convenient way to manage shared content, ensuring consistency without the need to duplicate or manually update information in each portal.

At the moment, you can add root folders (along with their subfolders) and root pages as Linked Folders and Pages.

It’s also important to note that you cannot add a folder to a portal if access to it has been restricted through [granular permissions](https://thefusebase.com/guides/client-portal/granular-permissions/).

## **Adding a Linked Folder or Page** [#](#0-toc-title)

1) In the customizer of the desired portal, click on the option to create a folder or page.

![](https://thefusebase.com/wp-content/uploads/2025/03/image-25-1024x543.png)

2) In the dialog that appears, select Linked.

![](https://thefusebase.com/wp-content/uploads/2025/03/image-26-1024x596.png)

3) Then, choose the portal where the required folder or page is located.

4) Next, select the specific page or folder you want to link.

![](https://thefusebase.com/wp-content/uploads/2025/03/image-27-1024x676.png)

Next, the folder or page will be added to your portal as Linked, and any changes made in the source portal will automatically sync across all portals where the linked folder or page exists.

![](https://thefusebase.com/wp-content/uploads/2025/03/image-28-1024x515.png)

It’s important to note that almost all linked folders and pages are available in read-only mode (with the exception of Blank/Rich blocks, which you can edit directly within the linked page).

So, if you need to add a page to a linked folder or edit a linked page, you’ll need to go to the source portal of that folder or page.

To navigate to the source portal, click on the desired page or folder, then in the control bar, click Open original. You will be redirected to the customizer of the source portal, where you can edit the page or folder.

![](https://thefusebase.com/wp-content/uploads/2025/03/image-29-1024x535.png)
