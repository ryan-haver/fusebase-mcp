# Import documents from Notion

## Export pages from Notion [#](#b4126738932_4)

-   Log into your **Notion account**.
-   Open the desired Workspace.
-   Select **Export** from the menu.

![Export pages from Notion](https://app.nimbusweb.me/box/attachment/5533621/io4i2i41hnzugshwcx9a/arP2z3KRDqolJQK8/Group%20218.png)

-   Select **HTML** and click on **Export**.

![Export pages from Notion](https://app.nimbusweb.me/box/attachment/5533621/io4i2i41hnzugshwcx9a/Omsxco7YQ4OAnVEN/Group%20211%20(1).png)

-   Save the exported file.

## Import pages to Nimbus [#](#b1311438826_3)

![Import pages to Nimbus](https://app.nimbusweb.me/box/attachment/5533621/io4i2i41hnzugshwcx9a/JcFgNryxP66xSmvb/notionn_3.gif)

-   Open [Nimbus Note](https://thefusebase.com/note/).
-   In the import menu, click on **Import from Notion**.

![Import pages to Nimbus](https://app.nimbusweb.me/box/attachment/5533621/io4i2i41hnzugshwcx9a/1VyGZLakpwpJ1NL9/Group%20212%20(1).png)

![Import pages to Nimbus](https://app.nimbusweb.me/box/attachment/5533621/io4i2i41hnzugshwcx9a/pQdDAkOpGZiNyvyx/Group%20209%20(2).png)

-   Select the exported **HTML file**.
-   Wait for the import to finish.
