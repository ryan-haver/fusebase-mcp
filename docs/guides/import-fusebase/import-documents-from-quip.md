# Import documents from Quip

## Export pages from Quip [#](#b512580804_1)

-   Log in to your **Quip account**.
-   Open the document you want.
-   Select **HTML** from the Document – **Export** menu.

![Export pages from Quip](https://app.nimbusweb.me/box/attachment/5533625/7of68nf4k6xhp6ywad8b/yeOPK2cvqSg6TULp/Group%20210%20(1).png)

-   Save the exported file.

## Import pages to Nimbus [#](#b2698483985_0)

![Import pages to Nimbus](https://app.nimbusweb.me/box/attachment/5533625/7of68nf4k6xhp6ywad8b/UOgYvBRUk4PSwKNr/quip%20(2).gif)

-   Open [Nimbus Note](https://thefusebase.com/note/).
-   On the import menu, click on **Import from Quip**.

![Import pages to Nimbus](https://app.nimbusweb.me/box/attachment/5533625/7of68nf4k6xhp6ywad8b/vmETA0Mof1CXqK5F/Group%20208%20(1).png)

![Import pages to Nimbus](https://app.nimbusweb.me/box/attachment/5533625/7of68nf4k6xhp6ywad8b/88Ac8cyZvnCgKFFF/Group%20209%20(2).png)

-   Select the exported **HTML file**.
-   Wait for the import to finish.
