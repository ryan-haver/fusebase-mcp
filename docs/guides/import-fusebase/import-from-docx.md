# Import from DOCX

In [Nimbus Note](https://thefusebase.com/note/), you can import documents from docx as a single file or the whole zip file at once.

-   Open Nimbus Note.
-   In the import menu, click on **Import from DOCX**.

![Import from DOCX](https://app.nimbusweb.me/box/attachment/7398331/olxyk892wdor2n5ze4o2/5bNX2O11DkHGtxYL/image.png)

You can also find the import block from the menu inside the load

![You can also find the import block from the menu inside the load](https://app.nimbusweb.me/box/attachment/7398331/olxyk892wdor2n5ze4o2/S51cS22LJ0kUOLP2/image.png)

-   Select the DOCX file or ZIP file that contains DOCX documents
-   Wait for the import to finish.
