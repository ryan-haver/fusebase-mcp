# How to add my own watermark to a video?

## How to add my own watermark to a video? [#](#b639820073_0)

To get started, you can watch this video showing basic Watermark features.

1) Before recording a video, press Add your watermark.

![Before recording a video, press Add your watermark.](https://thefusebase.com/images/woterenv1.jpg)

2) You will be taken to the Watermark settings window, where you need to press Enable watermark.

![You will be taken to the Watermark settings window, where you need to press Enable watermark.](https://thefusebase.com/images/wateren2.jpg)

3) Select watermark type:  
– if you are using a picture, select “file”

![Select watermark type](https://thefusebase.com/images/wateren4.jpg)

– if you are using text, write the one you need, then choose font and color.

![Select watermark type](https://thefusebase.com/images/wateren3.jpg)

4) Specify the watermark location on the video.

![Specify the watermark location on the video.](https://thefusebase.com/images/wateren5.jpg)

Now watermarks will be added to all of your videos automatically.
