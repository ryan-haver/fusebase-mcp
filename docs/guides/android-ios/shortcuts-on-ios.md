# Shortcuts on iOS

Shortcut is a quick way to get a task done with your apps.

[Nimbus Note](https://thefusebase.com/note/) for iOS allows you to quickly execute a specific action using shortcuts.

At the moment, you can create shortcuts for the following actions:

1.  Opening a workspace
2.  Opening the search screen
3.  Creating a page in a workspace
4.  Adding a new task to task list.

To create a shortcut follow the steps:

-   Open the Shortcut app.

![Nimbus Note for iOS allows you to quickly execute a specific action using shortcuts.](https://app.nimbusweb.me/box/attachment/6265917/01dyqekmotphddkzzb6v/jwdPTxi4EOyMe6Gs/image.png)

-   Tap on **+** button.

![Nimbus Note for iOS allows you to quickly execute a specific action using shortcuts.](https://app.nimbusweb.me/box/attachment/6265917/01dyqekmotphddkzzb6v/4JcA2M5YHbC4VPEQ/image.png)

-   Find the Nimbus app.

![Nimbus Note for iOS allows you to quickly execute a specific action using shortcuts.](https://app.nimbusweb.me/box/attachment/6265917/01dyqekmotphddkzzb6v/1ElFRs1b15CYYHai/image.png)

-   Select the desired action and specify the workspace/task list (depending on your choice).

![Nimbus Note for iOS allows you to quickly execute a specific action using shortcuts.](https://app.nimbusweb.me/box/attachment/6265917/01dyqekmotphddkzzb6v/1mOJjQsdh3zz13pI/image.png)

![Nimbus Note for iOS allows you to quickly execute a specific action using shortcuts.](https://app.nimbusweb.me/box/attachment/6265917/01dyqekmotphddkzzb6v/M0MwU1D82WgXp4DM/image.png)

-   Add the created shortcut to Home Screen.

![Nimbus Note for iOS allows you to quickly execute a specific action using shortcuts.](https://app.nimbusweb.me/box/attachment/6265917/01dyqekmotphddkzzb6v/lnixFN25EMNU28d2/image.png)

-   Your shortcut is now ready to use!

![Nimbus Note for iOS allows you to quickly execute a specific action using shortcuts.](https://app.nimbusweb.me/box/attachment/6265917/01dyqekmotphddkzzb6v/bm1LKpAyItePYWiE/image.png)

> We do not recommend adding “note” or “notes” wording to the name of the shortcut, as it can lead to conflicts with the current functionality of Apple.
