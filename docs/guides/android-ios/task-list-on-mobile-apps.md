# Task list on mobile apps

A task list is an element in a page that allows you to create tasks, add people to them and set labels & deadlines. Added tasks can be seen in a special dashboard for more convenient control over their execution.

In [Nimbus Note](https://thefusebase.com/note/) you can work with tasks even from your phone! Just add your tasks to the task list and easily manage them. You can change your task position in one task list or move it to another list. What’s more, you can sort all your tasks by labels, assignees and dates using a dashboard.

![A task list is an element in a page that allows you to create tasks, add people to them and set labels & deadlines.](https://app.nimbusweb.me/box/attachment/7324206/wzk9eig6io6kmm7ibp9o/F6Hu4RMYVUYPMLdb/960w_1b%202499%20tasks.gif)

## How to add a task list? [#](#b3665292979_259)

To create a task list, tap on the **Task list** icon in the panel inside the editor.

![To create a task list, tap on the Task list icon in the panel inside the editor](https://app.nimbusweb.me/box/attachment/7324206/wzk9eig6io6kmm7ibp9o/MbIhmMgosVFN3wqS/image.png)

To add a new task, click on Add new Task.

![To add a new task, click on Add new Task.](https://app.nimbusweb.me/box/attachment/7324206/wzk9eig6io6kmm7ibp9o/SSXDhNDQ6UhstPvd/image.png)

Enter the task title and tap on **Enter**. The task will be added to list –

![Enter the task title and tap on Enter. ](https://app.nimbusweb.me/box/attachment/7324206/wzk9eig6io6kmm7ibp9o/yKw1VG4tuGNr9zzD/image.png)

## How to change the name of the Task list? [#](#b2774059286_1173)

1) Tap on the task name – 

![Tap on the task name](https://app.nimbusweb.me/box/attachment/7971212/xx0ue4ds4mm7np2spk5a/C4K3kpMWNzeEkXOa/image.png)

2) Next, tap on the name of the task and enter a new name –

![Next, tap on the name of the task and enter a new name](https://app.nimbusweb.me/box/attachment/7971212/xx0ue4ds4mm7np2spk5a/BFtTGj0RlqGLbHAl/image.png)

Next, confirm the changes with a tap on enter.

## How to edit the task? [#](#b2774059286_993)

To edit a task, just click on it, make the necessary changes and tap Enter.

![How to edit the task?](https://app.nimbusweb.me/box/attachment/7324206/wzk9eig6io6kmm7ibp9o/puYVkc6WHwLKEX0z/image.png)

## How to mark a task as completed? [#](#b3665292979_3944)

To mark the task as completed, click on the circle next to the task –

![How to edit the task?](https://app.nimbusweb.me/box/attachment/7324206/wzk9eig6io6kmm7ibp9o/VVguqhQ8wbyznycM/image.png)

The completed task will be crossed out.

![How to edit the task?](https://app.nimbusweb.me/box/attachment/7324206/wzk9eig6io6kmm7ibp9o/gI8oyCkA76tYe9le/image.png)

## How to assign a task to a workspace member? [#](#b3665292979_1015)

-   To add responsible members to a task, click on the row with the task – the actions panel will appear.
-   Tap on the **Assignee.**

![How to assign a task to a workspace member?](https://app.nimbusweb.me/box/attachment/7324206/wzk9eig6io6kmm7ibp9o/xshKoPPq6aNDwLbK/image.png)

In the list that appears, select up to 3 workspace members that you want to assign to the task.

![In the list that appears, select up to 3 workspace members that you want to assign to the task.](https://app.nimbusweb.me/box/attachment/7324206/wzk9eig6io6kmm7ibp9o/Trv7YJ0U1RLdJPYJ/image.png)

The selected members’ avatars will be shown right under the task.

![The selected members' avatars will be shown right under the task.](https://app.nimbusweb.me/box/attachment/7324206/wzk9eig6io6kmm7ibp9o/PeHNQQgohJSCKkXE/image.png)

## How to add labels to tasks? [#](#b3665292979_1480)

To add labels to a task, tap on the **Label** on the actions panel.

![To add labels to a task, tap on the Label on the actions panel.](https://app.nimbusweb.me/box/attachment/7324206/wzk9eig6io6kmm7ibp9o/rH1FofHlsHS9mUvv/image.png)

In the window that appears, select as many labels as you want. You can also create a new label by entering its name into the search field and tapping the Create label button.

![In the window that appears, select as many labels as you want. You can also create a new label by entering its name into the search field and tapping the Create label button.](https://app.nimbusweb.me/box/attachment/7324206/wzk9eig6io6kmm7ibp9o/glRdcU1gS29fRq9p/image.png)

The selected labels will be shown under the task.

![The selected labels will be shown under the task.](https://app.nimbusweb.me/box/attachment/7324206/wzk9eig6io6kmm7ibp9o/VjY1lqQerT63ImiK/image.png)

## How to add a deadline to tasks? [#](#b3665292979_2553)

To add a deadline to a task, click on the **Due date** on the actions panel.

![To add a deadline to a task, click on the Due date on the actions panel.](https://app.nimbusweb.me/box/attachment/7324206/wzk9eig6io6kmm7ibp9o/VVNXnXvdJHAcdRMD/image.png)

You will see a pop-up, where you can choose a date –

![You will see a pop-up, where you can choose a date ](https://app.nimbusweb.me/box/attachment/7324206/wzk9eig6io6kmm7ibp9o/11iMWycgShJSQvQM/image.png)

If you want to see the time next to the date, enable the Show time toggle.

![If you want to see the time next to the date, enable the Show time toggle.](https://app.nimbusweb.me/box/attachment/7324206/wzk9eig6io6kmm7ibp9o/KiCtlIZyiJ9v0R9R/image.png)

You can click on time to change it.

![](https://app.nimbusweb.me/box/attachment/7324206/wzk9eig6io6kmm7ibp9o/MOjKrgGqH6nhZDxK/image.png)

![If you want to see the time next to the date, enable the Show time toggle.](https://app.nimbusweb.me/box/attachment/7324206/wzk9eig6io6kmm7ibp9o/yzyO7ls67aiEnV1T/image.png)

You can also add a reminder to the date. For this, select reminding options in the Alert block.

![You can also add a reminder to the date. For this, select reminding options in the Alert block.](https://app.nimbusweb.me/box/attachment/7324206/wzk9eig6io6kmm7ibp9o/n7lkwoP4AktqRgSy/image.png)

The selected date will be shown under the task.

![The selected date will be shown under the task.](https://app.nimbusweb.me/box/attachment/7324206/wzk9eig6io6kmm7ibp9o/zL4fOEEFcDBhWxDQ/image.png)

On the selected reminder time, all assigned members will receive a notification about the task.

You can read more about the reminders [here](https://thefusebase.com/guides/basics/inline-reminders/).

## How to change the position of a task? [#](#b3985444174_1486)

To change the position of a task, just tap and hold on the task and drag it to the desired position.

## Detailed view of the task [#](#b3209563875_236)

Each task can be opened in a special detailed view, where you can see detailed information about the task.

To go to the detailed view, you just need to tap on the name of the task –

![To go to the detailed view, you just need to tap on the name of the task](https://app.nimbusweb.me/box/attachment/7971212/xx0ue4ds4mm7np2spk5a/RW8oDCjvyXLlzZ0w/image.png)

Next, it will open in a detailed view, where you can interact with the task.

### Changing the task name [#](#b3209563875_571)

Tap on the task name to change its name. Use the tap to confirm –

![Tap on the task name to change its name. Use the tap to confirm](https://app.nimbusweb.me/box/attachment/7971212/xx0ue4ds4mm7np2spk5a/NUuTcCIXVnc61drw/image.png)

### Adding/modifying the task description [#](#b3209563875_694)

The task name doesn’t always give you a full understanding of the task. With a description, you can add more context to the task. To add a description, tap on its box and enter the text of the description –

![To add a description, tap on its box and enter the text of the description](https://app.nimbusweb.me/box/attachment/7971212/xx0ue4ds4mm7np2spk5a/72JAWVRpkVnasc9F/image.png)

### Adding/changing assignee for a task [#](#b3209563875_973)

In the detailed view, you will be able to specify those responsible for the task. Click on Assignee –

![In the detailed view, you will be able to specify those responsible for the task. Click on Assignee](https://app.nimbusweb.me/box/attachment/7971212/xx0ue4ds4mm7np2spk5a/R4dMMAncBah79Ye3/image.png)

And select the necessary participants in the window that appears, and then confirm the action by tapping on the check mark

Also in the detailed view, you can select labels for the task. Tap on Labels –

![Also in the detailed view, you can select labels for the task. Tap on Labels](https://app.nimbusweb.me/box/attachment/7971212/xx0ue4ds4mm7np2spk5a/nwrnIe8bxejYiyFB/image.png)

Long tap on the icon next to the comment and choose ‘Reply’- from the menu.

![](https://thefusebase.com/wp-content/uploads/2022/12/image-6-2.png)

Long tap your avatar next to your comment and choose ‘Edit’ – from the menu.

![](https://thefusebase.com/wp-content/uploads/2022/12/image-7.png)

Next, write a new response and click the submit button –

![](https://thefusebase.com/wp-content/uploads/2022/12/image-8.png)

#### How do I delete a comment? [#](#21-toc-title)

Long tap your avatar next to your comment and choose ‘Delete’ from the menu.

![](https://thefusebase.com/wp-content/uploads/2022/12/image-9.png)

Next, confirm the deletion of the comment.

## Where can I see all my tasks in one place? [#](#b1253715136_2)

In Nimbus Note app, you can see all tasks in one common Dashboard. To see it, just click on the Tasks icon in the bottom navigation block.

![In Nimbus Note app, you can see all tasks in one common Dashboard. To see it, just click on the Tasks icon in the bottom navigation block.](https://app.nimbusweb.me/box/attachment/7324206/wzk9eig6io6kmm7ibp9o/SqoXm5WuVJnx53vS/image.png)

You will see a general list of all your uncompleted tasks. Here you can also mark tasks as completed, assign responsible members, add labels and set deadlines, edit the task list & task name or add a new task (in the same way as you do from inside the page).

![You will see a general list of all your uncompleted tasks.](https://app.nimbusweb.me/box/attachment/7324206/wzk9eig6io6kmm7ibp9o/3eaGyl1OdNbrNJWx/image.png)

Completed and uncompleted tasks are separated in the dashboard. Use the tabs on the top to navigate between them.

![Completed and uncompleted tasks are separated in the dashboard. Use the tabs on the top to navigate between them.](https://app.nimbusweb.me/box/attachment/7324206/wzk9eig6io6kmm7ibp9o/TOJBFdwRemc81Bnj/image.png)

Using the task menu, you can:

-   Add a new task above the current one
-   Add a new task below the current one
-   Set labels to the task
-   Change assignee on the task
-   Set date
-   Duplicate the task
-   Delete the task

![](https://app.nimbusweb.me/box/attachment/7324206/wzk9eig6io6kmm7ibp9o/iqP3OJ46Ac6gwLVp/image.png)

If you want to navigate to the page, just tap on the task list name and select the Go to page option in the menu that appears.

![If you want to navigate to the page, just tap on the task list name and select the Go to page option in the menu that appears.](https://app.nimbusweb.me/box/attachment/7324206/wzk9eig6io6kmm7ibp9o/UDdsLksjVhrTHumw/image.png)

Also, this menu allows you to mark all task in the list as completed and rename the task list.

> On iOS, you can create a shortcut for quickly creating tasks. You can find more details [here](https://thefusebase.com/guides/android-ios/shortcuts-on-ios/).
