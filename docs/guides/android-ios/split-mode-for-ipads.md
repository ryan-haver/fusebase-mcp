# Split mode for iPads

Now you can use [Nimbus Note](https://thefusebase.com/note/) with other apps simultaneously. Just try Split mode for iPad. With it, you can easily work with two apps using one screen.

Moreover, you can interact with other apps. For example, you can drag and drop images from Unsplash directly into a page.

[Embedded content](https://www.youtube.com/embed/w-mVUrNH2rc?feature=oembed)

Also, you can work on two pages at the same time. Compare & edit two documents at the same time – and more.

All you need to do is:

-   Open a page.
-   Tap on the three dots on the top of the screen and select Split mode.
-   Open Nimbus one more time in the second part of the screen and choose the desired page.
