# Clipper for the mobile clients

In the [Nimbus Note](https://thefusebase.com/note/) app for Android and iOS, you can save information from the Internet using the built-in clipper.

-   Open the desired page in a browser.
-   Tap on the sharing icon –

![In the Nimbus Note app for Android and iOS, you can save information from the Internet using the built-in clipper.](https://app.nimbusweb.me/box/attachment/6443583/g0faaxuxqzgk9p9f6ytx/GGkvUB3dnLQd0bzE/image.png)

-   In the list of applications, select Nimbus –

![In the Nimbus Note app for Android and iOS, you can save information from the Internet using the built-in clipper.](https://app.nimbusweb.me/box/attachment/6443583/g0faaxuxqzgk9p9f6ytx/mN48QORaGapg8Nxq/image.png)

-   In the window that appears, you can select the desired workspace, folder and name of the future page –

![In the Nimbus Note app for Android and iOS, you can save information from the Internet using the built-in clipper.](https://app.nimbusweb.me/box/attachment/6443583/g0faaxuxqzgk9p9f6ytx/utB0rRfPf15VbvO4/image.png)

-   You can also choose how you want to save the link:
    -   As a bookmark
    -   As an article (the text + pictures from the page will be saved)
    -   As a regular link

![In the Nimbus Note app for Android and iOS, you can save information from the Internet using the built-in clipper.](https://app.nimbusweb.me/box/attachment/6443583/g0faaxuxqzgk9p9f6ytx/cdxUWMhyHUfeJrYZ/image.png)

-   After selecting the desired type of future page, tap on the save icon –

![In the Nimbus Note app for Android and iOS, you can save information from the Internet using the built-in clipper.](https://app.nimbusweb.me/box/attachment/6443583/g0faaxuxqzgk9p9f6ytx/GnlUp7iHW7VLzp17/image.png)

-   After processing, the page will be added to your Nimbus Note account.
