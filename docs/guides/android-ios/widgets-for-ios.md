# Widgets for iOS

In the iOS mobile app, you can add widgets of different types. The following widgets are currently available:

-   Quick notes widget
-   Notes list widget
-   A task list widget

These widgets will help you quickly interact with your notes and task lists, as well as make changes without directly opening the app.

## How to add a widget [#](#b2587456571_360)

Make a long tap on a blank space on the screen and press **the plus +**

![Make a long tap on a blank space on the screen and press the plus +](https://app.nimbusweb.me/box/attachment/7825448/02sypnucli0pquwufa7r/oE7sxhOTSqr7rRVB/%D0%B8%D0%B7%D0%BE%D0%B1%D1%80%D0%B0%D0%B6%D0%B5%D0%BD%D0%B8%D0%B5.png)

Find [Nimbus Note](https://thefusebase.com/note/) widgets and select the desired one.

![Find Nimbus Note widgets and select the desired one.](https://app.nimbusweb.me/box/attachment/7825448/02sypnucli0pquwufa7r/8WTDOeLfnCmakM3J/%D0%B8%D0%B7%D0%BE%D0%B1%D1%80%D0%B0%D0%B6%D0%B5%D0%BD%D0%B8%D0%B5.png)

## Notes widget [#](#b2587456571_530)

### Widget for quickly creating notes [#](#b2587456571_553)

Widgets for the quick creation of notes allow you to easily create the desired type of note without opening the application.

![Widgets for the quick creation of notes allow you to easily create the desired type of note without opening the application.](https://app.nimbusweb.me/box/attachment/7825448/02sypnucli0pquwufa7r/eveHrUrdf2OvVtEh/%D0%B8%D0%B7%D0%BE%D0%B1%D1%80%D0%B0%D0%B6%D0%B5%D0%BD%D0%B8%D0%B5.png)

### Notes list widgets [#](#b2587456571_730)

The notes list widget allows you to view a list of notes from any folder, as well as quickly navigate to a note by tapping on it. The widget allows you to configure in which folder notes will be shown.

![The notes list widget allows you to view a list of notes from any folder, as well as quickly navigate to a note by tapping on it. ](https://app.nimbusweb.me/box/attachment/7825448/02sypnucli0pquwufa7r/WLsqBkQIpTdpSB6j/%D0%B8%D0%B7%D0%BE%D0%B1%D1%80%D0%B0%D0%B6%D0%B5%D0%BD%D0%B8%D0%B5.png)

To select a folder, you need to make a long tap on the widget and select Edit widget. Next, you will be able to select a project and a folder.

Also, you can create a widget for individual pages. To do this, select Page widget. Then, in the widget settings, select the desired page.

![Also, you can create a widget for individual pages. ](https://app.nimbusweb.me/box/attachment/7825448/02sypnucli0pquwufa7r/UP3TSTH7MHfTt3Wf/%D0%B8%D0%B7%D0%BE%D0%B1%D1%80%D0%B0%D0%B6%D0%B5%D0%BD%D0%B8%D0%B5.png)

## Task list widget [#](#b2587456571_1259)

On iOS, you can create two kinds of task list widgets: small and large. With widgets you can:

-   view tasks from an individual task list
-   create new tasks.

### Compact task list widget [#](#b2587456571_1473)

The compact task list widget will not take up much space on your desktop, but it will allow you to quickly view tasks within one task list and go to the note by tapping on it. In the settings, you can specify which task list you want to see in the widget.

![In the settings, you can specify which task list you want to see in the widget.](https://app.nimbusweb.me/box/attachment/7825448/02sypnucli0pquwufa7r/YyzgCCXiZWGfOhMO/%D0%B8%D0%B7%D0%BE%D0%B1%D1%80%D0%B0%D0%B6%D0%B5%D0%BD%D0%B8%D0%B5.png)

### Large task list widget [#](#b2587456571_1772)

The large task list widget allows you to view tasks from the selected task list. In the widget settings, you can select the desired list and tasks and specify whether or not to show completed tasks in the widget.

![In the widget settings, you can select the desired list and tasks and specify whether or not to show completed tasks in the widget.](https://app.nimbusweb.me/box/attachment/7825448/02sypnucli0pquwufa7r/vSlWihENIgtkoe7l/%D0%B8%D0%B7%D0%BE%D0%B1%D1%80%D0%B0%D0%B6%D0%B5%D0%BD%D0%B8%D0%B5.png)
