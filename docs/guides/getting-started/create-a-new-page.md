# Create a new page

With [FuseBase](https://thefusebase.com/), you can easily create pages with different data and styles. You can also customize the appearance of pages with the help of our built-in tools and sidebar.

Every page in FuseBase is a unique area where you can add or embed almost anything. You can also [share created pages as Public Pages](https://thefusebase.com/guides/embedding-sharing/shared-folders-and-pages/) for anyone in the world to view.

Open your workspace in FuseBase and create pages. You can write down your ideas and add different information such as documents, links, audio and video recordings.  
Create different folders and subfolders. Easily store all important information and access it any time you want.

## A quick way to create a page [#](#b2170224716_657)

There are several ways to quickly create a page:

-   Click the + in the Navigation bar and select **New Page**. You can immediately choose the workspace in which the page will be created.

![](https://thefusebase.com/wp-content/uploads/2022/08/A-quick-way-to-create-a-page.png)

-   Click on the desired folder and press +. Next, select **New page**

![](https://thefusebase.com/wp-content/uploads/2022/08/select-New-page.png)

-   Go to **All Pages** or to any folder and click on the **New Page** button.

![](https://thefusebase.com/wp-content/uploads/2022/08/select-New-page-1-1024x391.png)

## Search through the template gallery for inspiration [#](#b2073301619_10)

We have ready-made templates for different categories. Try them and see for yourself how easy and convenient they are!

-   Create a new page.
-   Click on **Templates**.
-   Select the template you like and click **Use template**.

![](https://thefusebase.com/wp-content/uploads/2022/08/Use-template-1024x524.png)

## Access block options [#](#b3662530494_443)

It’s very simple and quic to work with blocks. Just select the menu, and an options list will appear.

-   Click on the three dots menu

![](https://thefusebase.com/wp-content/uploads/2022/08/Access-block-options-1024x526.png)

## Convert Blocks [#](#b2119842762_1497)

Convert blocks to code / embed / task / quote etc.

To do this:

-   Click on the three dots menu
-   Choose **Convert to**

![](https://thefusebase.com/wp-content/uploads/2022/08/Convert-Blocks-1024x497.png)

## More items [#](#b633710630_545)

-   Click on the arrow in the editor to display More items:
    -   Outline **cmd/ctrl** + **⌥/alt** **+** **o**
    -   Section break **cmd/ctrl** + **h**
    -   Quote **cmd/ctrl** + **⌥/alt** **+** **q**
    -   Link **cmd/ctrl** + **k**
    -   Code **cmd/ctrl** + **⌥/alt** **+** **c**
    -   Embed **cmd/ctrl** + **⌥/alt** **+** **m**
    -   Bookmark **cmd/ctrl** + **⌥/alt** **+** **b**
    -   Mention **@**
    -   Google Drive **cmd/ctrl** + **⌥/alt + g**

## Embed popular websites like YouTube, Google Maps, Google Docs [#](#b633710630_1121)

You can easily add embeds from popular sites. You can also use Google Sheets to work with spreadsheets.

-   Click the arrow and select **Embed/Integrations** or use **cmd/ctrl** + **⌥/alt + m**
-   Insert or print the URL

![](https://thefusebase.com/wp-content/uploads/2022/08/Embed.png)

**[Read more about Embed](https://thefusebase.com/guides/embedding-sharing/embed-insert-google-drive-files/)**

## Create Table [#](#b633710630_1469)

-   Select the table icon or use **cmd/ctrl** + **⌥/alt + t**

![](https://thefusebase.com/wp-content/uploads/2022/08/tables.png)

-   Pull the corner to resize the table

![](https://thefusebase.com/wp-content/uploads/2022/08/table.gif)

## Insert objects into your table [#](#b633710630_1702)

-   Click on the column menu and select **Column type**
-   Select a cell and use the **Cell type** menu

![](https://thefusebase.com/wp-content/uploads/2022/08/objects-into-your-table.gif)

## Set styles to your text [#](#9-toc-title)

Use different styles to format and change the appearance of your text.

-   Select text
-   Select **bold/italic/highlight**

## Expand pages to full-width [#](#b3662530494_2603)

-   Click on the three dots menu
-   Toggle the **full-width** setting on or off, or use **cmd/ctrl** + **alt + w**
