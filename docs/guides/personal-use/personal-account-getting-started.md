# Personal account | Getting Started

Nimbus Note is one place where you can manage all your information / knowledge bases / workspaces / tasks.

Transform information chaos from multiple sources into an organized workplace. Empower yourself or your team to get things done faster with less iterations and fewer meetings.

When creating a page, you are not only able to input text, but also use tables, lists, attachments, bookmarks, mentions, embeds, links and much more. Apply a variety of styles to format and change the appearance of your text.

[Nimbus Note](https://thefusebase.com/note/) is user-friendly and easy to navigate – see for yourself below!

## A quick way to create a page [#](#b2931399491_10)

-   Click on **+** or use the keyboard combination **ctrl** \+ **opt/alt** + **n** to create a page.

![A quick way to create a page](https://app.nimbusweb.me/box/attachment/4581502/1gs30a1nch1gx90qb5ml/uucM0FU5g1krt5Du/Group%201%20(10)%20(1).png)

### Lifehack [#](#b2574226151_10)

In Nimbus Note, you can create a page in a new tab:

-   Right-click on **+** button.
-   Select **Create page in new tab**

![In Nimbus Note, you can create a page in a new tab](https://app.nimbusweb.me/box/attachment/4581502/1gs30a1nch1gx90qb5ml/PeAz9ARS9zFMvFAx/Group%2096.png)

You can also open an existing page in a new tab:

-   Right-click on the page.
-   Select **Open in new tab**

![You can also open an existing page in a new tab](https://app.nimbusweb.me/box/attachment/4581502/1gs30a1nch1gx90qb5ml/5RXMkBNXNHHEqpBi/Group%2097.png)

## Easily create tables and set the number of rows and columns [#](#b952559027_427)

You can create tables, change the number of rows & columns and amend the appearance of the table.

-   Click on the table icon or use **cmd/ctrl** + **⌥/alt** + **t**.
-   Pull the corner to resize the table.

![Easily create tables and set the number of rows and columns](https://app.nimbusweb.me/box/attachment/4581502/1gs30a1nch1gx90qb5ml/g5EtV4PEW32afK1B/table_and_database%20(2).gif)

## Insert objects into your table [#](#b952559027_634)

-   Click on the column menu and select **Column type**.
-   Select a cell and use the **Cell type** menu.

![Insert objects into your table](https://app.nimbusweb.me/box/attachment/4581502/1gs30a1nch1gx90qb5ml/afGom0Jobzb9w50J/0%20(3).gif)

## Drop files and move blocks [#](#b952559027_796)

-   Drag and drop files into the editor.
-   Move files to columns using drag and drop.

![Drop files and move blocks](https://app.nimbusweb.me/box/attachment/4581502/1gs30a1nch1gx90qb5ml/NYB5ucTqpUQKBaIH/00%20(2).gif)

![Drop files and move blocks](https://app.nimbusweb.me/box/attachment/4581502/1gs30a1nch1gx90qb5ml/BXYUH7aqUJEY1svr/1444%20check.gif)

## More items [#](#b3942405283_10)

-   Click on the arrow in the editor to display More items:
-   outline
-   section break
-   quote
-   link
-   code
-   embed
-   bookmark
-   mention
-   Google Drive

![More items](https://app.nimbusweb.me/box/attachment/4581502/1gs30a1nch1gx90qb5ml/lD7pF4N9RuVGomL1/more_items%20(2).gif)

### Embed from popular services like YouTube, Google Maps and Google Docs [#](#b952559027_1617)

You can easily add embeds from popular sites. You can also use Google Sheets to work with spreadsheets.

-   Click on the arrow and select **Embed** or use **cmd/ctrl** \+ **⌥/alt** \+ **m**.
-   Paste or type the URL.

[Embedded content](https://www.youtube.com/embed/pawZgUt8Tu8?feature=oembed)

### Access block options [#](#b952559027_1924)

It’s very easy and convenient to work with blocks. Just select the menu, and a list with many options will appear.

-   Click on the three dots menu.

![Access block options](https://app.nimbusweb.me/box/attachment/4581502/1gs30a1nch1gx90qb5ml/z0x4PH3qowFkr54d/menu%20(2).gif)

### Convert Blocks [#](#b952559027_2121)

Convert blocks to code / embeds / tasks / quotes, and so on.

For this:

-   click on the three dots menu
-   choose **Convert to**

![Convert Blocks](https://app.nimbusweb.me/box/attachment/4581502/1gs30a1nch1gx90qb5ml/YWwIS0oGrGbUanQM/2020_06_18_13_10_50.gif)
