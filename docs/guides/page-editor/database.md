# Tables in FuseBase Document Editor – Quick Guide

You can convert columns and individual cells into different types for displaying and storing information. If you converted a whole column, the type it was converted to will be displayed every time a new line is created in that column.

![If you converted a whole column, the type it was converted to will be displayed every time a new line is created in that column.](https://app.nimbusweb.me/box/attachment/3630911/qo6i8l5cqik2yfk9o0tu/TCqrPjeDi6dBmdaJ/image.png)

For example, you can create your own task tracker, product accounting system, project manager, or mini CRM system.

To convert a column, click on the menu arrow and select Column type:

![To convert a column, click on the menu arrow and select Column type](https://app.nimbusweb.me/box/attachment/3630911/qo6i8l5cqik2yfk9o0tu/4Gpy5x0Ci4CDQMvc/image.png)

If you want to convert only a single cell, right-click on it and select Cell type:

![If you want to convert only a single cell, right-click on it and select Cell type](https://thefusebase.com/images/xRD2HAA9AP.gif)

Important! If the column has already been converted, you cannot convert the cells from that column to a different type.

Columns and cells can be converted to the following types:

## Multi type [#](#b470768785_477)

This is a default column. It can contain both text and numbers; you can also convert individual cells to other types in this column.

## Text [#](#b470768785_541)

You can enter different kinds of text information in the column cells.

![You can enter different kinds of text information in the column cells.](https://app.nimbusweb.me/box/attachment/3630911/qo6i8l5cqik2yfk9o0tu/tewj6eC3mHtgqpQk/image.png)

## Numbers [#](#b470768785_630)

You can enter data in the form of numbers in the column cells. These can include integers, decimals, or percentiles. You can change the format in the column or cell settings.

![You can enter data in the form of numbers in the column cells. ](https://app.nimbusweb.me/box/attachment/3630911/qo6i8l5cqik2yfk9o0tu/dMrhmBbi0IW9WXz6/image.png)

![You can enter data in the form of numbers in the column cells. ](https://thefusebase.com/images/bden13.jpg)

## Currency [#](#b470768785_650)

You can enter numbers with a currency symbol next to them. The dollar sign is set by default, but you can set the desired currency in the column or cell settings. There, you can also choose where to display the currency symbol – before or after the number.

![You can enter numbers with a currency symbol next to them. ](https://app.nimbusweb.me/box/attachment/3630911/qo6i8l5cqik2yfk9o0tu/rr3eptMIBE6NLUx9/image.png)

## Single select [#](#b470768785_929)

A single selection is perfect if you want a single parameter or option from the preset list. When editing a cell, an auto-complete menu will be suggested. You can pick the needed option from the drop-down menu or create a new one. To create a new one, just type its name and press Enter. You can also assign a color for each option. The options you do not need can be deleted.

![You can pick the needed option from the drop-down menu or create a new one. ](https://thefusebase.com/images/OJY8b9Ra5V.gif)

## Multiple select [#](#b470768785_955)

What makes multiple selection different from single selection is that a few options can be displayed in a single cell. You just need to select the needed options from the list (or enter new ones) and press Enter to confirm.

![What makes multiple selection different from single selection is that a few options can be displayed in a single cell.](https://thefusebase.com/images/xe1fBMJrnI.gif)

Also, you can choose the look of the label. There are 3 types of labels:

-   Filled.
-   No fill, but with a colored frame.
-   Plain text.

You can change the type and its color in the label settings. You can display different types of labels in one cell/column.

![You can change the type and its color in the label settings. You can display different types of labels in one cell/column.](https://app.nimbusweb.me/box/attachment/3630911/qo6i8l5cqik2yfk9o0tu/179p5LfcTNhV8XRV/4H5klUsVTS.gif)

## Progress bar [#](#b470768785_983)

The progress bar provides a special scale for marking progress (for example, when completing a task). You can mark progress using a 0 – 100% scale. By default, the bottom is set at 0%, but you can change it in the column or cell settings.

![The progress bar provides a special scale for marking progress (for example, when completing a task). You can mark progress using a 0 - 100% scale.](https://thefusebase.com/images/BSabpJYc3m.gif)

![The progress bar provides a special scale for marking progress (for example, when completing a task). You can mark progress using a 0 - 100% scale.](https://app.nimbusweb.me/box/attachment/3630911/qo6i8l5cqik2yfk9o0tu/PC4EuXi5q7TvBRim/image.png)

## Collaborator [#](#b470768785_1008)

In this cell, you can select one or more project participants from the list. You can mark a team member responsible for completing a certain task.

![You can mark a team member responsible for completing a certain task.](https://thefusebase.com/images/3Vyfi4OdhB.gif)

## Date and time [#](#b470768785_1033)

In this cell, you can specify the desired date and time, as well as easily edit it if necessary. You can set the date using the calendar. The date format can be chosen in the column settings.

![ You can set the date using the calendar. The date format can be chosen in the column settings.](https://app.nimbusweb.me/box/attachment/3630911/qo6i8l5cqik2yfk9o0tu/UnpnVHIQv9i3edOM/image.png)

If you need to set the time as well, please enable the needed option. You can select your preferred time format (e.g. 12h or 24h).

![If you need to set the time as well, please enable the needed option. ](https://app.nimbusweb.me/box/attachment/3630911/qo6i8l5cqik2yfk9o0tu/XngCpjZOQuThizhC/image.png)

## Rating [#](#b470768785_1059)

This cell is for rating certain actions or objects. By default, you can rate between 1 and 5, but this can be changed to 1 and 10 in the cell /column settings. You can also change the rating appearance, specifically the star symbols.

![This cell is for rating certain actions or objects. By default, you can rate between 1 and 5, but this can be changed to 1 and 10 in the cell /column settings.](https://app.nimbusweb.me/box/attachment/3630911/qo6i8l5cqik2yfk9o0tu/X43SMs6m5oQN5q3w/image.png)

![This cell is for rating certain actions or objects. By default, you can rate between 1 and 5, but this can be changed to 1 and 10 in the cell /column settings.](https://app.nimbusweb.me/box/attachment/3630911/qo6i8l5cqik2yfk9o0tu/7TfON3KbBn9CPwa3/image.png)

## Link [#](#b470768785_1078)

These cells are for quickly adding links. To do that, click inside the cell, and in the form that appears, enter the link and text (if needed). If you need to access the link, click on it and press Open. To change the link, click on it and press Change. You can also delete the link right away.

![To do that, click inside the cell, and in the form that appears, enter the link and text (if needed).](https://app.nimbusweb.me/box/attachment/3630911/qo6i8l5cqik2yfk9o0tu/ZQzLj7GSlfdUUOzG/image.png)

## Mentions [#](#b470768785_1095)

These cells are for quickly selecting and mentioning:

-   other participants of the project (in this case the participant will receive a notification)
-   another page
-   a folder
-   a workspace
-   date

![These cells are for quickly selecting and mentioning](https://app.nimbusweb.me/box/attachment/3630911/qo6i8l5cqik2yfk9o0tu/YoopX1dHbL4KBo5s/image.png)

To go to another page, folder, or workspace, just click on the link. There you can also change the cell.

![To go to another page, folder, or workspace, just click on the link. There you can also change the cell.](https://app.nimbusweb.me/box/attachment/3630911/qo6i8l5cqik2yfk9o0tu/hLYsb2EAsLGzVgtZ/image.png)

## Attachments [#](#b470768785_1143)

These cells are for adding files and images. You can add multiple files to a single cell. To add a file, click on the cell and press Upload, then select the needed files and wait for the upload to complete (you can stop the upload by clicking on X).

![You can add multiple files to a single cell. ](https://thefusebase.com/images/bden9.jpg)

To open or download a file, click inside the cell, then click on the required file.

![To open or download a file, click inside the cell, then click on the required file.](https://thefusebase.com/images/bden10.jpg)

If you want to rename or delete a file, click on the file menu and select desired option.

![If you want to rename or delete a file, click on the file menu and select desired option.](https://app.nimbusweb.me/box/attachment/3630911/qo6i8l5cqik2yfk9o0tu/96XWUzlN9WhAECAk/image.png)

## What does Reset cells to default mean when choosing column type? [#](#b470768785_1392)

If you change cell type in a column, to avoid conflicts between a single cell and a column, changing the column type comes with a limit. You first need to reset all converted cells and only then convert the column.

![If you change cell type in a column, to avoid conflicts between a single cell and a column, changing the column type comes with a limit.](https://app.nimbusweb.me/box/attachment/3630911/qo6i8l5cqik2yfk9o0tu/1yukiBIzvmgUj11x/image.png)

![If you change cell type in a column, to avoid conflicts between a single cell and a column, changing the column type comes with a limit.](https://app.nimbusweb.me/box/attachment/3630911/qo6i8l5cqik2yfk9o0tu/BkDpn3hK8ftirYJb/image.png)

## Can I create multiple lines in a text cell? [#](#b470768785_1917)

Yes, use Shift + Enter to create a new line.

## I don’t like that the table is taking up the full width of the screen. How to change that? [#](#b1267072681_205)

Open the table menu and in Table Layout uncheck Full width.
