# Reminders in Nimbus Web Clipper

### How to set a reminder in clipper? [#](#b2636885466_0)

1) Click on the reminder icon.

![How to set a reminder in clipper?](https://thefusebase.com/images/crem1.jpg)

2) Select a date and specify the time. Set the reminder to repeat if needed.

![Select a date and specify the time. Set the reminder to repeat if needed.](https://thefusebase.com/images/crem2.jpg)

3) Click Done.

### How to change a reminder? [#](#1-toc-title)

Click the reminder icon and set new parameters.

### How to delete a reminder? [#](#2-toc-title)

Click on the reminder icon and press Delete.

![How to delete a reminder?](https://thefusebase.com/images/crem4.jpg)
