# Embed Typeform

## How do I copy an embed code from Typeform? [#](#b1910415905_12)

-   Log in to your **Typeform** account or create a new account.
-   Select typeform and click on **Share**.

![How do I copy an embed code from Typeform?](https://app.nimbusweb.me/box/attachment/5584991/fsxns55ca904bxp83wz2/SY0VrEZgpmqBEn4V/Group%20228.png)

-   Go to **Embed in a web page section**.

![Go to Embed in a web page section.](https://app.nimbusweb.me/box/attachment/5584991/fsxns55ca904bxp83wz2/YEUm0HtEK02A7MYi/Group%20229.png)

-   Select the **embed mode** and press on **Get the code**.

![Select the embed mode and press on Get the code.](https://app.nimbusweb.me/box/attachment/5584991/fsxns55ca904bxp83wz2/LZGANzLoePYco58A/Group%20230.png)

-   **Copy the code.**

![Copy the code.](https://app.nimbusweb.me/box/attachment/5584991/fsxns55ca904bxp83wz2/dSsrYXgAqXeSkxev/Group%20231.png)

## Adding Typeform embed to FuseBase [#](#b1910415905_546)

-   Open [FuseBase](https://thefusebase.com/).
-   Click on the arrow and select **Embed** or **cmd/ctrl** + **⌥/alt** + **m**.

![](https://thefusebase.com/wp-content/uploads/2022/09/Group-232.png)

-   Add the code using **Cmd/Ctrl**+**V**.

![](https://thefusebase.com/wp-content/uploads/2022/09/Group-233-1024x579.png)

### Resize Embeds [#](#b1861605483_291)

-   Drag the upper-right corner and **resize** (all proportions will be preserved).

![](https://thefusebase.com/wp-content/uploads/2022/09/Group-234-1024x578.png)

-   For free resizing, hold the **ALT key** and start resizing using the bottom-right chip.

![](https://thefusebase.com/wp-content/uploads/2022/09/Group-235-1024x481.png)

**Note: To return the embed to its original position, use the hotkey Ctrl + Z, or click on the three dots and select Undo.**
