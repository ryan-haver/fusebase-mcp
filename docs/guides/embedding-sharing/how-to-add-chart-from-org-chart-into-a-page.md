# How to add chart from Org chart into a page in FuseBase

![How to add chart from Org chart into a page in Nimbus](https://app.nimbusweb.me/box/attachment/5180679/6ixz5ri1sng1fk604sco/6yTeCfqAU6JpcYdl/time.gif)

## How to copy the sharable link in Org Chart [#](#b30790726_529)

-   Open [https://orgchart.timeisltd.com/charts](https://orgchart.timeisltd.com/charts).
-   Click on **Share**.

![How to copy the sharable link in Org Chart](https://app.nimbusweb.me/box/attachment/5180679/6ixz5ri1sng1fk604sco/OLBrxs5Rp2cXkAal/Group%20184.png)

-   Select **Get sharable link**.
-   Under the Link to Org Chart section, select Copy.

![Under the Link to Org Chart section, select Copy.](https://app.nimbusweb.me/box/attachment/5180679/6ixz5ri1sng1fk604sco/5qZka0HpRwad3Alh/Group%20185.png)

## How to add chart from Org chart into a page in FuseBase [#](#b30790726_842)

-   Open [FuseBase](https://thefusebase.com/).
-   Insert the link through **right-click** or **ctrl**+**v**.
-   Select **Embed**.

![How to add chart from Org chart into a page in Nimbus](https://app.nimbusweb.me/box/attachment/5180679/6ixz5ri1sng1fk604sco/kwxKOJuIOHu4pYFb/Group%20186.png)

### Resize Embeds [#](#b2916505007_0)

-   Drag the upper-right corner and **resize** (all proportions will be preserved).

![Drag the upper-right corner and resize (all proportions will be preserved).](https://app.nimbusweb.me/box/attachment/5180679/6ixz5ri1sng1fk604sco/sZNdFyHWL7EzAdJE/Group%20187.png)

-   For free resizing, hold the **ALT** key and change the size using the bottom-right chip.

![For free resizing, hold the ALT key and change the size using the bottom-right chip.](https://app.nimbusweb.me/box/attachment/5180679/6ixz5ri1sng1fk604sco/9RqkKdSgwVa2zqml/Group%20188.png)

**Note: To return the embed to its original position, use the hotkey Ctrl + Z, or click the three dots and select Undo.**
