# Embed Miro

![Embed Miro](https://app.nimbusweb.me/box/attachment/5004037/bm25wapvb55t3o9qcllc/XXtL4qxYnJkBLwye/miro8_1.gif)

## How do I copy an embed link from Miro? [#](#b3636086686_0)

-   Click on the **Export** this board icon.

![How do I copy an embed link from Miro?](https://app.nimbusweb.me/box/attachment/5004037/bm25wapvb55t3o9qcllc/2bsLS7naXWSOKD2W/Group%20134.png)

-   Select **Embed**.

![How do I copy an embed link from Miro?](https://app.nimbusweb.me/box/attachment/5004037/bm25wapvb55t3o9qcllc/S8998ETXNM0bQotI/Group%20133.png)

-   Click on **Share Settings** and select the desired permissions.

![Click on Share Settings and select the desired permissions.](https://app.nimbusweb.me/box/attachment/5004037/bm25wapvb55t3o9qcllc/s3wd3tiSabNJprX2/Group%20132.png)

![Click on Share Settings and select the desired permissions.](https://app.nimbusweb.me/box/attachment/5004037/bm25wapvb55t3o9qcllc/FkFwg04SkxP7OtaY/Group%20131%20(1).png)

-   Select **Copy board link** and the link will be successfully copied.

![Select Copy board link and the link will be successfully copied.](https://app.nimbusweb.me/box/attachment/5004037/bm25wapvb55t3o9qcllc/aDoEBaxJ4BNYkuWN/Group%20130.png)

## Adding Miro embed to FuseBase [#](#b3107533274_0)

-   Open [FuseBase](https://thefusebase.com/).
-   Paste the link into the page by using **Cmd/Ctrl**+**V**, or by right-clicking and selecting **Paste**.
-   Select **Embed**.

![Adding Miro embed to Nimbus Note](https://app.nimbusweb.me/box/attachment/5004037/bm25wapvb55t3o9qcllc/GHcZnnWo0FnoZndv/miro7.gif)

### Resize Embeds [#](#b702295368_44)

-   Drag the upper-right corner and resize (all proportions will be preserved).

![Drag the upper-right corner and resize (all proportions will be preserved).](https://app.nimbusweb.me/box/attachment/5004037/bm25wapvb55t3o9qcllc/I582pNpJsfrTT0Fn/Group%20129.png)

-   For free resizing, hold the ALT key and change the size using the bottom-right chip.

![For free resizing, hold the ALT key and change the size using the bottom-right chip.](https://app.nimbusweb.me/box/attachment/5004037/bm25wapvb55t3o9qcllc/B7hGEVFEP00sDLO9/Group%20128.png)

**Note: To return the embed to its original position, use the hotkey Ctrl + Z, or click the three dots and select Undo.**

![To return the embed to its original position, use the hotkey Ctrl + Z, or click the three dots and select Undo.](https://app.nimbusweb.me/box/attachment/5004037/bm25wapvb55t3o9qcllc/BaUrhTh56mBkmr2Y/Group%20127.png)
