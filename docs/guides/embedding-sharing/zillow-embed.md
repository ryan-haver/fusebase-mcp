# Zillow embed

In this guide, we’ll show you how to add a search widget into your page from the popular real estate website Zillow.

## How to copy embed code from Zillow? [#](#b3735233272_25)

-   Go to [https://www.zillow.com/](https://www.zillow.com/)
-   Click on **LARGE ZILLOW SEARCH BOX**.

![Click on LARGE ZILLOW SEARCH BOX.](https://app.nimbusweb.me/box/attachment/5421170/zpt9sdez76s99xhy1usi/nnXl29cH6r8ghRH3/Group%20176%20(1).png)

-   Specify the desired **location**, for example New York, NY.

![Specify the desired location, for example New York, NY.](https://app.nimbusweb.me/box/attachment/5421170/zpt9sdez76s99xhy1usi/dFQXnCuI1TtK7Ix9/Group%20177%20(1).png)

-   **Copy** the widget code.

![Copy the widget code.](https://app.nimbusweb.me/box/attachment/5421170/zpt9sdez76s99xhy1usi/M60h9SvxGEg68VMI/Group%20178%20(1).png)

## How to insert an embed code into FuseBase [#](#b3454847724_9)

-   Open the [FuseBase](https://thefusebase.com/) and select **Embed**.

![How to insert an embed code into Nimbus Note](https://app.nimbusweb.me/box/attachment/5421170/zpt9sdez76s99xhy1usi/5hzFwG6JR9o3lMUn/Group%20179%20(1).png)

-   **Paste** the Zillow code into the embed.
-   Excellent! Search form added successfully.

![Paste the Zillow code into the embed.](https://app.nimbusweb.me/box/attachment/5421170/zpt9sdez76s99xhy1usi/BKVSgIpJKCx3uS81/Group%20181%20(1).png)
