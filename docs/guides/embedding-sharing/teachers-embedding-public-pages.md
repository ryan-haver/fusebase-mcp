# Teachers | Embedding public pages

Many Learning Management Systems (LMSs) set a limit on the size or type of information that you can input. [FuseBase](https://thefusebase.com/) allows you to create public pages in which you can insert any information without worrying about its size or what is inside it. You will save time, as all the changes made in the page will automatically fall into your embed on the site. After you create a page and make it public, use the Embed function in the same way that you would insert a YouTube video.

## You can embed public pages into your website [#](#b1733396173_1)

Open a public page/folder and copy the embed to your site

![You can embed public pages into your website](https://app.nimbusweb.me/box/attachment/4467748/byho6v6vmkmq16t0270o/jzuCqp6ieaxnBXBC/embed_new_1%20(2).gif)

**☝Note: To share a page / folder you need to:**

-   Right-click on the page / folder, select **Get a public link** -> **Copy link** -> **Done**

![Right-click on the page / folder, select Get a public link -> Copy link -> Done](https://app.nimbusweb.me/box/attachment/4467748/byho6v6vmkmq16t0270o/hnDQ0gfwhP5NatfI/public_note.gif)

-   open the shared page / folder and press on the **Share** icon
-   click on **<>** and copy the code

-   also, you can right-click on the page and select **Get a public link**
-   then switch to **Embed code**

![also, you can right-click on the page and select Get a public link
](https://thefusebase.com/wp-content/uploads/2022/08/Group-1-18-1024x567.png)

## Access to embeds by password [#](#b417920808_791)

Safety first! If the page/folder is protected with a password, then when you open the site with an embed, the content will be hidden, and you will need to enter the password.

![Safety first! If the page/folder is protected with a password, then when you open the site with an embed, the content will be hidden, and you will need to enter the password.](https://app.nimbusweb.me/box/attachment/4467748/byho6v6vmkmq16t0270o/LqkdkaJHH4iDa173/pass_new_1.gif)

-   When creating a public page, you will have access to the password protection field where you can specify the password you need, that is, when you open your site with an embed you will need to enter the password

![When creating a public page, you will have access to the password protection field where you can specify the password you need, that is, when you open your site with an embed you will need to enter the password](https://thefusebase.com/wp-content/uploads/2022/08/Image-10-1-1024x621.png)
