# Adding Microsoft OneDrive Document embeds to FuseBase

If you create Microsoft Office documents through OneDrive, then you will likely be interested in adding embeds of created documents to [FuseBase](https://thefusebase.com/).

## How to copy embed code in Microsoft OneDrive? [#](#b3959649356_68)

-   Open [https://onedrive.live.com](https://onedrive.live.com) through your account.
-   Hover the mouse over the desired document and click on the selecting icon.

![How to copy embed code in Microsoft OneDrive?](https://app.nimbusweb.me/box/attachment/5359327/bhhwpcipnhg59qmsvz2m/XZ6Dy3WWriNn8tPc/Group%20156%20(1).png)

-   Click on **Embed**.

![Click on Embed.](https://app.nimbusweb.me/box/attachment/5359327/bhhwpcipnhg59qmsvz2m/PiWeWqkP5sOlcDk5/Group%20157%20(1).png)

-   Click and copy the **iframe code**.

![Click and copy the iframe code.](https://app.nimbusweb.me/box/attachment/5359327/bhhwpcipnhg59qmsvz2m/TQqK7G4QL97DoN1p/Group%20158%20(1).png)

## How to add embed code into FuseBase? [#](#b3959649356_685)

-   Go to FuseBase and open the desired page.
-   Create an **embed object** and paste the **Iframe code**.
-   Done! The document is added to the page.

### Resize Embeds [#](#b3959649356_989)

-   Drag the upper-right corner and **resize** (all proportions will be preserved).

![Drag the upper-right corner and resize (all proportions will be preserved).](https://app.nimbusweb.me/box/attachment/5359327/bhhwpcipnhg59qmsvz2m/rEz5k2H3NYboVWSN/Group%20159%20(1).png)

-   For free resizing, hold the **ALT key** and change the size using the bottom-right chip.

![For free resizing, hold the ALT key and change the size using the bottom-right chip.](https://app.nimbusweb.me/box/attachment/5359327/bhhwpcipnhg59qmsvz2m/5XVov2PnPh5TdD7L/Group%20160%20(1).png)

**Note: To return the embed to its original position, use the hotkey Ctrl + Z, or click the three dots and select Undo.**
