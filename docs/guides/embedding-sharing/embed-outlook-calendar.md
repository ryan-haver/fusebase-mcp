# Embed Outlook Calendar

## How do I copy an embed code from Outlook? [#](#b2824420973_9)

If you use an Outlook calendar, you can add it to your [FuseBase](https://thefusebase.com/) as an embed. To do this, follow these simple steps:

-   Open your Outlook account in [https://outlook.live.com](https://outlook.live.com).
-   Press on **Settings** icon on the top right corner.

![If you use an Outlook calendar, you can add it to your Nimbus Note as an embed.](https://app.nimbusweb.me/box/attachment/5856036/96pbc775o7959g7l4g6c/9XeFw2tAg3fx6ahD/image.png)

Press on **View all Outlook Settings**.

![Press on View all Outlook Settings.](https://thefusebase.com/wp-content/uploads/2022/09/Frame-2147207153-13-1024x899.png)

-   Click on **Calendar** – **Shared calendars** and scroll to **Publish a calendar** –

![Click on Calendar - Shared calendars and scroll to Publish a calendar -](https://app.nimbusweb.me/box/attachment/5856036/96pbc775o7959g7l4g6c/P1kF9EBuM0ShpgdT/image.png)

-   Select the calendar that you want to add to the page, then choose the desired permission level and click on **Publish** –

![Select the calendar that you want to add to the page, then choose the desired permission level and click on Publish ](https://app.nimbusweb.me/box/attachment/5856036/96pbc775o7959g7l4g6c/YZOBqCHrJ4eZaMLq/image.png)

-   Click on the HTML link and select **Copy link** to copy it to the clipboard –

![Click on the HTML link and select Copy link to copy it to the clipboard -](https://app.nimbusweb.me/box/attachment/5856036/96pbc775o7959g7l4g6c/PQbmDH1VETHNfSdY/image.png)

## Adding Outlook embed to FuseBase [#](#b2015415409_9)

-   Click on the arrow and select **Embed** or use the **cmd/ctrl** \+ **⌥/alt** + **m** keyboard combination.

![](https://thefusebase.com/wp-content/uploads/2022/09/image-20-1-1024x619.png)

-   Paste the link to the calendar in the embed field.
-   The calendar will appear in your page!

![Paste the link to the calendar in the embed field.](https://thefusebase.com/wp-content/uploads/2022/09/Image-21-1-1024x603.png)

### Resize Embeds [#](#b762375094_0)

-   Drag the upper-right corner and **resize** (all proportions will be preserved).

![Drag the upper-right corner and resize (all proportions will be preserved).](https://thefusebase.com/wp-content/uploads/2022/09/Image-23-1-1024x442.png)

-   For free resizing, hold the **⌥/alt** key and start resizing using the bottom-right chip.

![For free resizing, hold the ⌥/alt key and start resizing using the bottom-right chip.](https://thefusebase.com/wp-content/uploads/2022/09/Frame-2147207152-20-1024x479.png)

**Note: To return the embed to its original position, use the hotkey Command/Ctrl + Z, or click on the three dots and select Undo.**
