# Embed Airtable

## How do I copy an embed code from Airtable? [#](#b1888568803_465)

You can add tables and databases of the [https://airtable.com](https://airtable.com/) service to [FuseBase](https://thefusebase.com/), and it’s very easy to do.

-   Open the desired table at [https://airtable.com](https://airtable.com/).
-   Click on **Share view**.

![How do I copy an embed code from Airtable?](https://app.nimbusweb.me/box/attachment/6034573/653plbhau3181hgq45gb/WnOpTTwiiUtdwAKU/image.png)

-   Click on **Embed this view on your site**.

![Click on Embed this view on your site.](https://app.nimbusweb.me/box/attachment/6034573/653plbhau3181hgq45gb/6FHcBMyVkcBPFvWT/image.png)

-   Customize the appearance and copy the embed code.

![Customize the appearance and copy the embed code.](https://app.nimbusweb.me/box/attachment/6034573/653plbhau3181hgq45gb/dHcHV7xYc1NP5RyH/image.png)

## Adding Airtable embed to FuseBase [#](#b1888568803_587)

-   Open the desired page in FuseBase.
-   Click on the arrow and select **Embed** or use the **cmd/ctrl** + **⌥/alt** + **m** keyboard combination.

![](https://thefusebase.com/wp-content/uploads/2022/09/image-15-2-1024x707.png)

-   Paste the copied code from Airtable.
-   The database will appear in your page!

![](https://thefusebase.com/wp-content/uploads/2022/09/image-16-4-1024x304.png)

### Resize Embeds [#](#b1888568803_1062)

-   Drag the upper-right corner and **resize** (all proportions will be preserved).

![](https://thefusebase.com/wp-content/uploads/2022/09/image-17-1.png)

-   For free resizing, hold the **⌥/alt** key and start resizing using the bottom-right chip.

![](https://thefusebase.com/wp-content/uploads/2022/09/image-18-2.png)

**Note: To return the embed to its original position, use the hotkey Command/Ctrl + Z, or click on the three dots and select Undo.**
