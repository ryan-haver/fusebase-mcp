# Personal | Protect Embeds via password

You can keep your data safe and secure by using a password on pages / folders.

## Protect Embeds via password [#](#b1509526342_0)

-   If you want to share sensitive information on public pages such as blogs or websites, you can set a password to the Embed and it will be accessible only to those with the password.

![If you want to share sensitive information on public pages such as blogs or websites, you can set a password to the Embed and it will be accessible only to those with the password.](https://app.nimbusweb.me/box/attachment/4475453/hzyl8ukv9yal8tvmlu7h/5doEeoNx0biHJcrd/2020_06_28_22_58_18.gif)

-   When creating a public page, you will have access to the Password protection field. Here you can specify the password you need. When you open your site with an embed, you will need to enter the password.

![When creating a public page, you will have access to the Password protection field.](https://thefusebase.com/wp-content/uploads/2022/08/image-21-2.png)
