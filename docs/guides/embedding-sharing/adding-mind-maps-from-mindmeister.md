# Adding Mind Maps from MindMeister

In this guide, we will explain how to add Mind Maps to pages using a well-known service to demonstrate: [https://www.mindmeister.com](https://www.mindmeister.com/).

## How to copy Embed code from MindMeister? [#](#b1971555835_788)

-   Go to [https://www.mindmeister.com](https://www.mindmeister.com) and log into your account (or create a new one).
-   Open the desired **Mind Map**.
-   Click on **Share**.

![How to copy Embed code from MindMeister?](https://app.nimbusweb.me/box/attachment/5421812/2n46cerym6optzokk22m/y1ZkbmNnIDkuRKnE/Group%20185%20(1).png)

-   Click on **Publish map**.

![Click on Publish map.](https://app.nimbusweb.me/box/attachment/5421812/2n46cerym6optzokk22m/ll29AIcqIrRvDOOT/Group%20186%20(1).png)

-   Click on **Embed map**.

![Click on Embed map.](https://app.nimbusweb.me/box/attachment/5421812/2n46cerym6optzokk22m/U2TISFHvV9dUhWe2/Group%20187%20(1).png)

-   Specify settings and **copy the code**.

![Specify settings and copy the code.](https://app.nimbusweb.me/box/attachment/5421812/2n46cerym6optzokk22m/J2XGTuVOc9aJF0sB/Group%20188%20(1).png)

## How to insert an embed code into FuseBase [#](#b1971555835_857)

-   Go to [FuseBase](https://thefusebase.com/) and open the page you want.
-   Select **Embed** and **paste the code** with MindMeister.

![How to insert an embed code into Nimbus Note](https://app.nimbusweb.me/box/attachment/5421812/2n46cerym6optzokk22m/Tdg1tcqbBxa0eoaI/Group%20189%20(1).png)

-   Everything is ready – Mind Map is added to the page!

![Everything is ready - Mind Map is added to the page!](https://app.nimbusweb.me/box/attachment/5421812/2n46cerym6optzokk22m/Orb6ONv0pdAxRIVO/Group%20190%20(1).png)
