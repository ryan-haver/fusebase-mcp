# Getting started with Branding

In [FuseBase](https://thefusebase.com/), you can change the appearance of public pages / folders and add different elements to your brand (logo, favicon, etc.). You can also customize the appearance of the web client (disable unnecessary items in Quick links, add your links to Quick Links, etc.)

This option is available only for organizations. For each workspace, you can change the appearance of public pages and the web client.

Deliver your services professionally with no mention of third-party tools with FuseBase’s no-code white-label solution:

-   Customize theme and colors
-   Add your brand name and logos
-   Set up branded system emails
-   Add a custom CNAME
-   Choose from 2000+ integrations

## Branding for public pages [#](#b2282531627_880)

You can change the appearance of public pages / folders and add components of your brand to them:

-   Switch on/ off page indexing by search engines
-   Disable/enable the top bar
-   Show/hide the copy link icon
-   Show/hide table of contents
-   Change Theme (white/dark)
-   Set the logo, specify the positioning, add the logo link, and set the favicon
-   Change the header settings.

You can do this through the business organization dashboard at [https://teams.nimbusweb.me](https://teams.nimbusweb.me/).

-   Go to the Workspaces page.
-   Click the menu of the desired workspace and select **Branding : Public pages.**
