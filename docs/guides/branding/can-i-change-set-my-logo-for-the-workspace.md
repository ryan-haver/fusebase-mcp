# Can I change/set my logo for the workspace?

1) Open the organization control panel.

2) Go to the list of workspaces.

![Go to the list of workspaces.](https://app.nimbusweb.me/box/attachment/4080931/bk8trab1r981msdpx5n3/JYsmvhUtAYAQB9GS/%D0%B8%D0%B7%D0%BE%D0%B1%D1%80%D0%B0%D0%B6%D0%B5%D0%BD%D0%B8%D0%B5.png)

3) Click on desired workspace.

4) Hover over the worspace logo and click on Change picture.

5) Select a picture from a computer system or add utilizing Drag and Drop.

6) Select the place in the image that you want to display in the logo. You can likewise change the size of the crop.

![](https://thefusebase.com/wp-content/uploads/2022/10/image-12-2.png)

7) Click on the checkmark to save the changes.

![](https://thefusebase.com/wp-content/uploads/2022/10/image-10.png)

![](https://thefusebase.com/wp-content/uploads/2022/10/image-11-1.png)

8) If you want to erase the logo, then again point to the logo and select Reset, and then confirm the action.
