# Adding CTA on public pages using Convertflow

In this article, we’ll show you how to add marketing tools to public pages by using the Converflow service.

-   Create an account on [https://www.convertflow.com/](https://www.convertflow.com/) or log in to an existing one.
-   Try adding Sticky bar to your page. To do this, click on the appropriate option.

![Try adding Sticky bar to your page. To do this, click on the appropriate option.](https://app.nimbusweb.me/box/attachment/5330717/l7f4oci3ks0x473degem/GiJtci44wVJPlNne/Group%20141%20(1).png)

-   Select the **desired bar**, for example, the following:

![Select the desired bar, for example, the following:](https://app.nimbusweb.me/box/attachment/5330717/l7f4oci3ks0x473degem/5oyoOSxkk8yA37wO/Group%20142%20(2).png)

-   Click on **Use templates**.

![Click on Use templates.](https://app.nimbusweb.me/box/attachment/5330717/l7f4oci3ks0x473degem/I3wR2lbCnfSG07yo/Group%20143%20(1).png)

-   Customize the bar according to your requirements and click on **Preview**.

![Customize the bar according to your requirements and click on Preview.](https://app.nimbusweb.me/box/attachment/5330717/l7f4oci3ks0x473degem/F6qK0qT3gTGS5tbd/Group%20144%20(1).png)

-   If everything suits you, then click **Launch**.

![If everything suits you, then click Launch.](https://app.nimbusweb.me/box/attachment/5330717/l7f4oci3ks0x473degem/zcc2p8Bhr6ndkAI3/Group%20145%20(1).png)

-   Now you need to **activate the bar** in a special switcher.

![Now you need to activate the bar in a special switcher.](https://app.nimbusweb.me/box/attachment/5330717/l7f4oci3ks0x473degem/TfZWpbMNvMraYM0u/Group%20147%20(1).png)

-   After activation, you will see a **special code** that needs to be added to FuseBase.

![After activation, you will see a special code that needs to be added to Nimbus.](https://app.nimbusweb.me/box/attachment/5330717/l7f4oci3ks0x473degem/cLIcplhEEZLva5F8/Group%20148%20(1).png)

-   Open the [FuseBase](https://thefusebase.com/) business console – [https://teams.nimbusweb.me/.](https://teams.nimbusweb.me/.)
-   Go to the desired workspace and select Custom **HTML / JS** from the menu.

![Go to the desired workspace and select Custom HTML / JS from the menu.](https://app.nimbusweb.me/box/attachment/5330717/l7f4oci3ks0x473degem/DnFDgbwXIXe14tto/Group%20149%20(1).png)

![Go to the desired workspace and select Custom HTML / JS from the menu.](https://app.nimbusweb.me/box/attachment/5330717/l7f4oci3ks0x473degem/ENNsryltDrTgsKJ3/Group%20150%20(1).png)

-   Click on **Add HTML / JS** and add the code from Convertflow to the <head> section and click on **Publish**.

![Click on Add HTML / JS and add the code from Convertflow ](https://thefusebase.com/wp-content/uploads/2022/09/Group-151-1-594x1024.png)

-   Now you can check how it looks!  
    To do this, just open any public page from the workspace and check for the availability of the bar.

![To do this, just open any public page from the workspace and check for the availability of the bar.](https://app.nimbusweb.me/box/attachment/5330717/l7f4oci3ks0x473degem/GvKmVxBSQ3gIdhxU/Group%20152%20(1).png)

Congratulations on the successful results!
