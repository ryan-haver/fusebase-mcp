# Adding Google Analytics to public pages using custom scripts

With the help of custom scripts, you can connect Google Analytics to get timely statistics about the use of your public pages.

-   Go to the organization management panel – [https://teams.nimbusweb.me](https://teams.nimbusweb.me/).
-   Go to workspace management.
-   Click on **Custom HTML / JS** in the workspace menu.

![Click on Custom HTML / JS in the workspace menu.](https://app.nimbusweb.me/box/attachment/4581241/ien1k6syzz18uwuy7b2v/SCPr1AYYrNRj5qWH/Group%20207%20(1).png)

-   Press on **Add HTML/JS**.

![Press on Add HTML/JS.](https://app.nimbusweb.me/box/attachment/4581241/ien1k6syzz18uwuy7b2v/rFnXda6NPxxacgth/Group%20206%20(1).png)

-   Open [https://analytics.google.com](https://analytics.google.com) under your account.
-   Go to account management and click on **Create property** (also, you can use an already created Property).

![Go to account management and click on Create property (also, you can use an already created Property).](https://app.nimbusweb.me/box/attachment/4581241/ien1k6syzz18uwuy7b2v/1gp8JSUvDTxyt9uN/Group%20205%20(1).png)

-   Choose **Web** and press on **Continue**.

![Choose Web and press on Continue.](https://app.nimbusweb.me/box/attachment/4581241/ien1k6syzz18uwuy7b2v/e8OOMjmRfAlqVw0x/Group%20204%20(1).png)

-   Specify the website name and link to your subdomain (be sure to select https).

![Specify the website name and link to your subdomain (be sure to select https).](https://app.nimbusweb.me/box/attachment/4581241/ien1k6syzz18uwuy7b2v/UwEpRWR2I7f5doEG/Group%20203%20(1).png)

-   Then click on **Tracking info** and choose **Tracking code**.

![Then click on Tracking info and choose Tracking code.](https://app.nimbusweb.me/box/attachment/4581241/ien1k6syzz18uwuy7b2v/DSZfOqH81D3xL2yB/Group%20202%20(1).png)

-   In the page that appears, copy the code.

![In the page that appears, copy the code.](https://app.nimbusweb.me/box/attachment/4581241/ien1k6syzz18uwuy7b2v/atNYV6JBUjt5eXNh/Group%20201%20(1).png)

-   Return to the FuseBase organization console. Add the resulting code to the <head> </head> field and click **Publish**.

![Return to the Nimbus Note organization console. ](https://thefusebase.com/wp-content/uploads/2022/08/Adding_Google_Analytics-1.png)

Analytics connected successfully!
