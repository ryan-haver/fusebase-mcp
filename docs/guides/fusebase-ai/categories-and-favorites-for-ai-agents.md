# Categories and Favorites for AI Agents

If you have many [AI agents](https://thefusebase.com/guides/fusebase-ai/ai-agents-quick-guide/), it might take a lot of time to find the right one. To avoid this, we’ve added support for categories and Favorites.

Currently, we only support predefined categories, but we’ll soon add the ability to create your own categories.

## Changing the category for an AI agent [#](#0-toc-title)

You can select a category when creating or editing an agent. To do this, click on the category input and choose the desired category. One agent can belong to multiple categories at the same time.

You can also select categories through the agent menu.

## Favorites [#](#1-toc-title)

Favorites is a special category where you can add your most frequently used agents. You can add an agent to Favorites through the agent menu. Clicking again removes the agent from Favorites.
